# CUDA-LBM 圆柱绕流项目环境总结

## 硬件

| 项目 | 信息 |
|------|------|
| GPU 架构 | Ada Lovelace（RTX 4060） |
| Compute Capability | 8.9 |
| 显存 | 8.6 GB |
| nvcc 编译参数 | `-arch=sm_89` |

---

## 软件环境

| 项目 | 信息 |
|------|------|
| 操作系统 | WSL2（Ubuntu） |
| Python 管理 | Miniconda，环境名 `lbm_cuda` |
| Python 版本 | 3.13 |
| 已装 Python 库 | numpy、scipy、matplotlib、numba |
| CUDA 驱动位置 | Windows 宿主机侧（WSL 通过桥接访问） |
| 编辑器 | VS Code + Remote WSL 扩展 |

## 物理参数（Re=100 圆柱绕流）

| 参数 | 符号 | 值 |
|------|------|----|
| 雷诺数 | Re | 100 |
| 圆柱直径 | D_phys | 0.1 m |

## GPU 版本可用规模（8.6 GB 显存）

D2Q9，float32，双缓冲内存估算：

| D_lat | 域大小 | 显存占用 | 说明 |
|-------|--------|---------|------|
| 20 | 400×400 | ~18 MB | CPU 验证版 |
| 40 | 800×800 | ~72 MB | GPU 推荐起点 |
| 80 | 1600×1600 | ~288 MB | 高精度 |
| 200 | 4000×4000 | ~1.8 GB | 显存充裕，可用 |



## 常用命令

```bash
# 激活环境
conda activate lbm_cuda

# 编译（单文件）
nvcc -arch=sm_89 -O3 src/main.cu -o lbm_sim

# 编译（多文件）
nvcc -arch=sm_89 -O3 src/main.cu src/lbm_kernel.cu src/force.cu -o lbm_sim

# 运行
./lbm_sim

# 后处理
python postprocess/analyze.py
```
